## 背景

原计划用 Supabase PostgreSQL，但 Supabase 存在问题。改用 Docker 本地 PostgreSQL 时，Docker 被 SOCKS5 代理拦截无法拉镜像（Docker Hub / gcr.io / 阿里云镜像均超时）。

## 方案：sql.js（SQLite → WASM → 浏览器）

- 使用 sql.js npm 包，SQLite 编译为 WASM，浏览器内运行
- 数据持久化到 localStorage（每次写操作后序列化整个 DB）
- 接口与 PostgreSQL 版本完全一致，后期迁移只需换实现
- 零网络依赖、零安装
